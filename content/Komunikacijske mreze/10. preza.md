# World Wide Web
- osnovni zahtjev - pristup informacijama koje su međusobno povezane hipermedijskim vezama
- dodatno služi i kao sučelje za e-mail, ftp i druge usluge poput internet bankarstva
- klijent - poslužitelj
- **hipertekst** - dio teksta koji omogućuje skok na neko drugo mjesto u dokumentu ili drugi dokument
	- hipermedij - hipertekst koji sadrži i slike, zvukove, video i druge medije
- **HTML**  - markup jezik koji služi za jednostavan zapis teksta i nudi mogućnost umetanja hiperveza
	- najnovija verzija je HTML5
- **URI** - Uniform Resource Identifier - jedinstveni identifikator resursa na mreži
	- **URL** - Uniform Resource Locator - dio URI-ja koji sadrži i lokaciju resursa
		npr. `https://www.fer.hr`
	- **URN** - Uniform Resource Name - dio URI-ja koji sadrži samo ime resursa, bez lokacije
		npr. `urn:fer:intranet
		- koristan jer omogućuje identifikaciju resursa bez obzira na njegovu lokaciju, znači ako se lokacija promjeni URN ostaje isti
- **HTTP** - protokol koji se koristi za prijenos podataka
	- zahtjevi:
		- **GET** - zahtjev za dohvat resursa
		- **POST** - zahtjev za slanje podataka na server
		- **PUT** - zahtjev za ažuriranje resursa
		- **DELETE** - zahtjev za brisanje resursa
		- **HEAD** - zahtjev za dohvat zaglavlja resursa
		- ...
	 - odgovori:
		 - **1xx** - informativni odgovori
			 - npr. `100 Continue`
		 - **2xx** - uspješni odgovori
			 - npr. `200 OK`
		 - **3xx** - odgovori koji upućuju na preusmjeravanje
			 - npr. `301 Moved Permanently`
		  - **4xx** - klijentske greške
			  - npr. `404 Not Found`
		  - **5xx** - serverske greške
			  - npr. `500 Internal Server Error`
	- primjer:
		- **HTTP komunikacija (klijent-poslužitelj)**:
			1. proces `www-poslužitelja` osluškuje TCP zahtjeve na portu 80
			2. korisnik upisuje URI adresu u preglednik
			3. preglednik saznaje IP adresu putem DNS upita
			4. uspostavlja se TCP veza s poslužiteljem (port 80)
			5. preglednik šalje HTTP zahtjev poslužitelju
			6. poslužitelj vraća HTTP odgovor s dokumentom
			7. TCP veza se zatvara
			8. preglednik prikazuje HTML dokument korisniku
	- **TCP** - protokol koji se koristi za prijenos podataka preko interneta
		- port 80, port 443 za HTTPS
		- jamči isporuku podataka
		- koristi se za slanje HTTP zahtjeva i odgovora
- **web klijenti** - programi koje koristimo za prikaz web stranica
	- preglednici (eng. browsers) - npr. Firefox, Chrome, Safari
	- botovi koji samostalno pretražuju web stranice (eng. web crawlers)

# Elektronička pošta

- omogućuje slanje i primanje poruka putem osobnih elektroničkih poštanskih adresa
- klijent-poslužitelj
	- klijent  (Mail User Agent - MUA) - program koji se koristi za čitanje, pisanje i slanje pošte 
		- npr. Thunderbird, Outlook, Gmail
	- poslužitelj (Mail Transfer Agent - MTA) - program koji se koristi za slanje i primanje pošte
- opći format e-mail adrese: `korisnik@poslužitelj.domena`
- format poruke:
	- zaglavlje (eng. header) - sadrži informacije o pošiljatelju, primatelju, predmetu i datumu slanja
    - tijelo (eng. body) - sadrži samu poruku
    - prilozi (eng. attachments) - datoteke koje se šalju uz poruku (MIME)
- **MIME** - Multipurpose Internet Mail Extensions - standard koji omogućuje slanje različitih vrsta datoteka putem e-maila
	- omogućuje slanje slika, zvukova, video zapisa, kompresiranih datoteka i drugih datoteka
    - MIME media type i multipart

## protokoli:
- **SMTP** - Simple Mail Transfer Protocol
	- protokol koji se koristi za slanje e-mail poruka
	- pošiljatelj šalje naredbu, primatelj odgovara kodom uspjeha/pogreške
- **POP3** - Post Office Protocol 3
	- protokol koji se koristi za dohvaćanje e-mail poruka s poslužitelja (dohvati i obriši)
	- sigurnosni problem: login credentials se šalju u običnom tekstu neenkriptirano
	- rješenje: Authenticated POP - APOP, Secure Sockets Layer - SSL/TLS
- **IMAP** - Internet Message Access Protocol
	- složeniji i napredniji protokol od POP3
	- poruke ostaju na poslužitelju
	- podržava dohvaćanje MIME dijelova poruka
	- omogućuje upravljanje mapama i pretraživanje poruka
	- potpora za šifriranje komunikacije
